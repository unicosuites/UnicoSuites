# Unico Suites

The official website for **Unico Suites Ltd** — premium serviced accommodation management in Bedford and the surrounding region.

🌐 **Live:** https://unicosuites.com

## Stack
- Static HTML / CSS / vanilla JavaScript (no framework, no build step).
- Hosted on **Netlify** with auto-deploy on push to `main`.
- Forms via **Netlify Forms**.
- Analytics via **GA4** (snippet activates only after cookie consent).

## Local development
```bash
cd site
python3 -m http.server 8080
# → http://localhost:8080
```

Or any equivalent static server (`npx serve`, `caddy file-server`, etc.). No install or build needed.

## Project structure
See [`CLAUDE.md`](CLAUDE.md) for the full file map, design tokens, conventions, deployment notes, and deal-analyser calculation model.

## Image pipeline
Source images live in `../public/`. The site assets in `assets/` are generated by:
```bash
python3 ../scripts/process_images.py
```
This runs `rembg` (AI background removal) on the founder portraits and produces circular crops + responsive WebP/JPEG sizes for every property photo and brand asset. Re-run after replacing any source image.

## Deployment
1. Push to `main`.
2. Netlify rebuilds (no build command — pure static).
3. `_headers` sets cache and security headers; `_redirects` handles the `co.uk → .com` 301 and the `/deal-analyser` short URL.
